What is html?

HTML is Hyper Text Markup Language first announced in 1991. HTML is now in HTML5.2 as the latest update came in in 2021

HTML has few features that you can use to format your own website.

Using h1 to h6 gives you different sized titles. Do not mistake the title as what you can see on top of the web browser as the name of the website.

What do you need in your html form?

You need the base of a html form. It will contain html, head, title and body. This is the bare minimum required. It will run with just html start and end tags, but it is not recommended. I will link below my go to html base that I use everytime I start doing a new html page.

SitePoint basic html boiler plate

When using links u will use html tag called A. Basic link setup is 'a href="http://xxxxxxxx"'.

If you need a bigger line change you can use BR tag. It will give you what ever big line change you want. You can customize the size of it in you stylesheet.

Images
